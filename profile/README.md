# Mimir

**A digital companion for tabletop RPG game masters.**

Mimir is a desktop application that helps Dungeon Masters run D&D 5e campaigns. It provides tools for managing campaigns, characters, encounters, and reference material—all in one place, all offline.

## Features

- **Campaign Management** — Organize your campaigns, sessions, and notes
- **Character Tracking** — Manage PCs and NPCs with full 5e stat blocks
- **Source Library** — Browse and search your owned rulebooks
- **AI Assistant** — Natural language tools for quick lookups and character creation
- **Offline First** — Your data stays on your machine
